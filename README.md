# 🎓 College Enquiry Chatbot

A smart web-based chatbot that helps students get instant answers to college-related queries such as courses, fees, admission process, facilities, and more.

---

## 🚀 Features

* 💬 Interactive chatbot interface
* 🎯 Answers queries related to:

  * Courses & syllabus
  * Fee structure
  * Admission details
  * College facilities
  * General FAQs
* 🧠 Uses Machine Learning / NLP concepts to understand user queries
* 🔐 User authentication system (Login/Register)
* 📩 Suggestion/feedback system
* 🌐 User-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python
* **Framework:** Flask / Django (based on implementation)
* **Database:** SQLite / MySQL
* **Other:** Machine Learning / NLP

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/SrujanaPhadke/College-Enquiry-Chatbot.git
cd College-Enquiry-Chatbot
```

### 2️⃣ Create Virtual Environment (Optional but Recommended)

```bash
python -m venv venv
source venv/bin/activate     # Linux/Mac
venv\Scripts\activate        # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```

### 5️⃣ Open in Browser

```
http://127.0.0.1:5000/
```

## 🧠 How It Works

1. User enters a query in natural language
2. The system processes the query using NLP / ML techniques
3. Matches it with predefined intents or dataset
4. Returns the most relevant response

This reduces manual enquiry workload and provides instant information to students.
